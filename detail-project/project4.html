Cashflow Management System Web App

A web application for managing cash flow and financial transactions with a modern design and user friendly interface

Project Overview

This project is a web application designed to manage cash flow and financial transactions. It provides a user-friendly interface for users to input, track, and analyze their financial data. The application includes features such as transaction management, reporting, and data visualization to help users understand their financial situation better.

Technology Used

Laravel Bootstrap CSS Chart JS MySQL Datatables
